# The Smesh Borger Joint 🍔

A UFC-themed burger shop opened by Khabib Nurmagomedov after his UFC 229 victory. Built with vanilla HTML, CSS, and JavaScript.

---

## How to Use

### Opening the App
Open `index.html` in any modern browser. No server, no dependencies, no installation required.

---

### Building a Burger

There are three ways to add ingredients:

- **Click** any ingredient button in the builder section
- **Drag** an ingredient card from the menu section (or a button) and drop it onto the burger
- **Keyboard shortcuts** — press a key while not typing in an input field:

| Key | Ingredient |
|-----|-----------|
| `T` | Tomato 🍅 |
| `O` | Onion 🧅 |
| `L` | Lettuce 🥬 |
| `C` | Cheese 🧀 |
| `P` | Patty 🥩 |
| `Esc` | Clear burger |

Each ingredient can be added up to **3 times**. Click any ingredient in the burger stack to remove it.

---

### Burger Actions

- **Khabib's Special** — loads the champion combo (patty + cheese + tomato + lettuce) instantly
- **Clear** — resets the current burger without affecting your basket
- **Name your burger** — type a custom name or click **🎲 Generate** for a UFC-themed auto-name based on your ingredients
- **Add to Basket** — saves the current burger (with name and price) to your order list

The **Nutrition Facts** panel on the right updates live as you add or remove ingredients.

---

### Your Basket

Added burgers appear as an itemized receipt showing the burger name (if set), ingredients as emojis, and individual price.

**Promo codes** — enter a code in the discount field and click Apply:

| Code | Discount |
|------|---------|
| `SMESH` | 15% off |
| `UFC229` | 10% off |
| `KHABIB` | 20% off |
| `HASBULLAH` | 5% off |
| `SUBMIT` | 25% off |

**Checkout** — clears the basket, fires a confetti burst, and plays a victory fanfare. Your basket is saved to `localStorage` and survives a page refresh.

---

### Extras

- **🥊 Smash Mode** — the floating red button in the bottom-right corner. Press it. (3-second cooldown.)
- **About page** — navigate to `about.html` for the lore behind Khabib's burger empire.
- All interactions play **sound effects** generated by the Web Audio API (no audio files needed).
